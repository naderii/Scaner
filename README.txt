# Network Scanner

این پروزه برای اسکن ای پی ها و پورت های باز آن در شببکه ایجاد شده است
## پیش‌نیازها

برای اجرای این پروژه، به موارد زیر نیاز دارید:
- Python 3
- pip (Python package installer)
- بسته‌های `python-nmap` و `tkinter`

## نصب

برای نصب پیش نیاز ها به فایل اینیستال دسترسی اجرا داده سپس ان را اجرا نماید
## اجرا
بعد از نصب پیش نیاز ها فایل پایتون را اجرا و بعد از باز شدن برنامه ای پی شبکه مورد نظر را وارد و دکمه  استارت اسکن را بزنید و منتظر بمانید تا اسکن تمام شود

## تشکر
این برنامه تقدیم به استاد خوبم خانم مهندس صحرایی بابت آموزش زبان پایتون میگردد
